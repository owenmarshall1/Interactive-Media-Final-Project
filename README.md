# Penumbra

Penumbra is a third-person survival horror game set in a graveyard and a heretical church. The player must solve puzzles and defeat enemies while managing a cigarette timer that serves as health. Enemies guard items and puzzles forcing the player to handle them.

---

## How to run

1. Go to the [Releases](https://github.com/owenmarshall1/Interactive-Media-Final-Project/releases) page
2. Download `Penumbra.exe` from the latest release
3. Run `Penumbra.exe`

No installation required. Windows only.

---

## Controls

| Action | Keyboard | Controller |
|---|---|---|
| Move | WASD | Left stick |
| Look | Mouse | Right stick |
| Aim | Q | LT |
| Shoot | Space | RT |
| Interact | E | A button |
| Use item | F | X button |
| Next item | 1 | RB |
| Previous item | 2 | LB |
| Pause | Escape | Start |

Controller recommended.
